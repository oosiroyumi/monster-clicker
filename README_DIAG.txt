# 診断ページの使い方

1. この `diag.html` を GitHub Pages の **リポジトリ直下**にアップロードします（`index.html` と同じ階層）。
2. ブラウザで `https://<あなた>.github.io/monster-clicker/diag.html?v=1` を開きます。
3. 画面の「クリック動作テスト」「setTimeout テスト」「Storage テスト」「Audio テスト」を順に押します。
4. 右側のログをコピーして、チャットに貼ってください。

## 判定の目安
- **JS: NG** → ページ自体のスクリプトが実行できていません。リポジトリの公開設定や HTML 内の構文に問題がある可能性があります。
- **Storage: NG** → `localStorage` が使えない環境です。ゲーム v4.1 以降ならメモリ保存に自動フォールバックしますが、古い版に戻っていないか確認してください。
- **Audio: NG** → 音が鳴らないだけで、ゲームの入力には通常影響しません。

> なお、ゲーム本体の `index.html` は `<script src="main.js" defer></script>` で読み込んでいます。`main.js` の 404 やキャッシュ差し替え漏れがあると「ボタンが反応しない」症状になります。`?v=fix1` のようなクエリを付けてリロードしてください。
